# プライバシーポリシー（一問から身につく SQL）

本リポジトリは、iOS アプリ「一問から身につく SQL」の **プライバシーポリシー** を公開するためのページです。

## 閲覧先

- **プライバシーポリシー**: [privacy-policy.html](privacy-policy.html)
- GitHub Pages 等で公開する場合は、公開後の URL（例: `https://<ユーザー名>.github.io/<リポジトリ名>/privacy-policy.html`）を App Store やアプリ内に記載してください。

## 用途

- App Store 審査時に「プライバシーポリシー URL」として提出するため
- ユーザーがアプリのデータ取り扱いを確認するため
